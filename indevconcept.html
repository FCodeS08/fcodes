Learn About Databases

What is a Database & SQL?

A database is an organized collection of structured information, or data, usually stored electronically and accessed using a database management system (DBMS). A SQL is a standardized programming language designed for managing and manipulating relational databases, stands for Structured Query Language.

Types of Databases

  • Relational Databases (MySQL, PostgreSQL, SQL Server)
  • NoSQL Databases (MongoDB, Redis)
  • Graph Databases (Neo4j)
  • Cloud Databases (Firebase, AWS DynamoDB)
